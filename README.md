# Trabalho de Tópicos que consome uma api já pronta

# Requisitos
	Ter java 8 instalado
	Ter Android Studio instalado
	Ter uma API rodando que permita um crud
# Instruções:
	1) Clonar o projeto
	2) Abrir o projeto no Android Studio
	3) Abrir o arquivo: java\consumer\IUveiculo
	4) Alterar a baseURL para o local onde a API está rodando
	5) Emular a aplicação em um emulador
